# Setup Guide

## Prerequisites

- **Python**: 3.11 or higher
- **Virtual Environment**: recommended (`venv`, `conda`)
- **Docker** (optional): for containerized deployment

## Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/example/ghost-sentry.git
    cd ghost-sentry
    ```

2.  **Initialize Environment**
    ```bash
    python -m venv .venv
    source .venv/bin/activate
    ```

3.  **Install Dependencies**
    ```bash
    pip install -e ".[dev]"
    ```

## Usage

### 1. Run a Mock Detection
Test the pipeline with sample imagery (included).

```bash
python -m ghost_sentry.cli data/samples/mock_detections.json --mock
```

### 2. Start the Backend API
Start the FastAPI server to stream results.

```bash
uvicorn ghost_sentry.api:app --reload
```
*Access API Docs at `http://localhost:8000/docs`*

### 3. Launch Operator Console
Run the Textual-based TUI.

```bash
python -m ghost_sentry.console.app
```

## Docker Deployment

Build and run the containerized service.

```bash
docker build -t ghost-sentry .
docker run -p 8000:8000 ghost-sentry
```
