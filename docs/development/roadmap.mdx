# Roadmap

## Phase 1: MVP (Complete) ✅
- [x] Optical imagery processing (Sentinel-2).
- [x] YOLOv8 tactical asset detection.
- [x] Lattice-compatible data modeling.
- [x] Autonomous cueing logic.
- [x] CoT XML output for interoperability.
- [x] TUI Operator Console.

## Phase 2-4: Core Capabilities (Complete) ✅
- [x] **SAR Integration**: Sentinel-1 support for all-weather monitoring.
- [x] **Edge Fusion**: Localized sensor fusion (Optical + Radar).
- [x] **Live Data**: Integration with real-time satellite feeds.

## Phase 5: Advanced Autonomy (Complete) ✅
- [x] **Event Bus**: Decoupled pub/sub architecture.
- [x] **Multi-Platform Cueing**: Support for cross-asset tasking.
- [x] **Behavior Analysis**: Loitering and anomaly detection.

## Phase 6: Operational Dashboards (Complete) ✅
- [x] **Task State Machine**: Full lifecycle tracking (Pending -> Completed).
- [x] **Tactical Console**: Web-based Command & Control interface.
- [x] **Real-Time Feeds**: WebSocket integration for live tracks.
- [x] **Asset Management**: Fleet utilization and status headers.

## Phase 7: Optimization & Planning (Complete) ✅
- [x] **Asset Telemetry**: Real-time position updates for friendly assets.
- [x] **Mission Planning**: Interactive map tools for defining AOs and paths.
- [x] **Assignment Optimization**: Multi-criteria logic (Fuel, Distance, Payload).
