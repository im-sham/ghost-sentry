# Workflow: Feature Development

1.  **Read Plan**: Check `.agent/artifacts/implementation_plans/detailed_execution_plan.md`.
2.  **Scope Lock**: Do not touch files outside the current phase.
3.  **Implement**: Write code + tests.
4.  **Verify**: Run the code/tests.
5.  **Log**: Update `walkthrough.md` (if verifying) or `active_context.md` (if handing off).
